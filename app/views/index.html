Prototypes for specific updates to the 'as is' application

Version 1 - alternative address

Assumption: Not everyone will know what address their benefit is registered to, so gathering where they live and letters sent will be enough to match addresses in our systems.

  • Start page
  • Start question about whether you're the applicant
  • Address question around assumption above

Start - version 1.0

Version 2 - alternative address

Assumption: Everyone knows what address their benefit is registered to, so asking for that address will be enough to match addresses in our systems, and then offer them an alternate correspondence address.

  • Start page
  • Start question about whether you're the applicant
  • Address question around assumption above

Start - version 2.0

Version 3

Assumption: Disregarded

Start - version 3.0

Version 4 - summary page

Assumption: People will be more likely to check their answers and read the declaration if these are on the same page, as the the former must be done to accept the latter.

  • Same content as version 1
  • Check answers showing full range of possible questions with declaration on same page
  • Back link added to each page

Start - version 4.0

Version 5 - summary page

Assumption: People will be more likely to check their answers and read the declaration if these are split over two pages, as otherwise there's too much to read and process at once.

  • Same content as version 1
  • Check answers on its own page
  • Declaration on its own page
  • Back link added to each page

Start - version 5.0

Version 6 - find address by postcode

Assumption: People will be more likely to enter the correct address that can be processed by robotics (and not fail) if they enter the essential parts and the system automatically fills in the rest.

  • Postcode and house number fields for lookup
  • Select address from those returned
  • Enter address manually if stuck or address not returned

Start - version 6.0

Version 7 - savings amount

Assumption: If the amount of savings is recorded in the application then they don't have to contacted to find out this figure, speeding up the application process.

  • Savings amount if 'yes' is selected

Start - version 7.0
